# 行为准则

## 我们的承诺

为营造开放、友善的环境，我们承诺：无论年龄、体型、残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教信仰或性取向如何，参与本项目的每个人都享有无骚扰的体验。

## 我们的标准

有助于营造积极环境的行为包括但不限于：

- 使用友好、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员保持同理心

参与者不可接受的行为包括但不限于：

- 使用性化语言或图像，以及不受欢迎的性关注或搭讪
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息（如地址、邮箱等）
- 其他在专业环境中可被合理认定为不当的行为

## 责任与执行

项目维护者有权利也有责任删除、编辑或拒绝与行为准则不符的评论、提交、代码、Wiki 编辑、Issue 及其他贡献，并可暂时或永久禁止其认为行为不当、威胁、冒犯或有害的参与者。

本行为准则适用于项目空间以及在与项目相关的公开场合代表项目时的行为。代表项目的情形可由维护者进一步界定。

## 适用范围

本准则适用于所有贡献者与维护者，以及在项目及其公开渠道中与社区互动的人员。

## 执行与反馈

若你认为有人违反本准则，或有其他疑虑，欢迎通过 Issue 或与维护者私下联系反馈。所有投诉将被认真对待并酌情处理。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
